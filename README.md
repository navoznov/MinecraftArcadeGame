# Minecraft Arcade Game

Браузерная аркада в стиле Minecraft: управляй Стивом, топчи зомби и проходи уровни через трубу из Марио.

## Запуск

Никакой сборки не нужно. Просто открой файл в браузере:

```
open index.html
```

## Управление

| Действие | Клавиатура | Геймпад (Xbox) |
|----------|-----------|----------------|
| Идти влево | ← | Левый стик / D-pad ← |
| Идти вправо | → | Левый стик / D-pad → |
| Прыгнуть | Пробел | Кнопка A |
| Рестарт / следующий уровень | Пробел / Enter | Кнопка A |
| Пауза | P | — |
| Снять паузу | P / ← → / Пробел | — |
| Открыть / закрыть инвентарь | Q | — |

## Как играть

- Прыгай на зомби сверху, чтобы их уничтожить
- Боковое касание зомби — мгновенная смерть
- После 5 убийств появляется зелёная труба из Марио — дойди до неё, чтобы пройти уровень
- На каждом следующем уровне зомби становятся быстрее и появляются чаще

## Архитектура

Весь код — в одном файле `index.html`. HTML, CSS и игровая логика встроены inline, без внешних зависимостей и сборщиков.

Подробнее о внутреннем устройстве — в [CLAUDE.md](CLAUDE.md).

## Changelog

### Инвентарь
- Клавиша **Q** открывает и закрывает инвентарь; клик за пределами панели тоже закрывает его
- При открытом инвентаре игра ставится на паузу; фоновая музыка продолжает играть
- Панель инвентаря в стиле Minecraft: тёмные слоты с рельефными рамками, серый фон
- Сетка **3×3** — девять слотов для предметов; изначально заполнена мечом, киркой и яблоком
- Слот **«Hand»** для экипировки в руку; клик по слоту сетки перемещает предмет в руку (и обратно)
- Иконки предметов нарисованы пиксельной графикой в стиле Minecraft

### Шахтные уровни
- Каждый 3-й и 4-й уровень в цикле (3, 4, 7, 8, 11, 12, ...) — подземная шахта
- Фон: тёмный каменный пещерный грот со случайными каменными блоками и прожилками угольной руды
- Платформы: деревянные доски с разделителями и текстурой волокна вместо травяных блоков
- Земля: каменный пол с блочной сеткой вместо травы и земли
- Освещение: 5 факелов на стенах и потолке с тёплым оранжевым свечением
- Без солнца, луны, звёзд и облаков; фантомы в шахтах не появляются

### Враги — Фантомы
- На ночных уровнях между первым и вторым ярусами платформ пролетают фантомы из Майнкрафта
- Пиксельный спрайт: тёмно-синие крылья, зелёные глаза, плавное покачивание по вертикали
- Фантомы спавнятся с той же частотой, что и зомби, но со сдвигом по фазе (~полцикла)
- Касание фантома — мгновенная смерть; убить его нельзя
- Фантомы исчезают на дневных уровнях и сбрасываются при переходе на следующий уровень

### Визуальное окружение
- Чередование дня и ночи по уровням: нечётные уровни — день (голубое небо, солнце, облака), чётные — ночь (тёмное небо, звёзды, луна с кратерами)

### Уровни и прогрессия
- Добавлена система уровней: после 5 убийств появляется труба из Марио
- При входе в трубу — экран «Level Complete» и переход на следующий уровень
- На каждом уровне зомби на 15% быстрее, интервал спавна на 15% короче
- После появления трубы зомби перестают спавниться

### Зомби
- Зомби с пиксельным спрайтом (зелёная кожа, светящиеся глаза)
- Механика топтания: прыжок сверху убивает зомби, боковое касание — убивает игрока
- Счётчик очков и HUD с номером уровня
- Исправлен баг: стартовые зомби больше не появляются в одной точке

### Звук (Web Audio API)
- Звук прыжка: синусоида 660→220 Гц
- Звук уничтожения зомби: квадратная волна 350→80 Гц
- Фоновая музыка: тема из Super Mario Bros. (Ground Theme) на квадратных волнах, зацикленная
- Звук смерти: три нисходящих «пиу» (440→330→220 Гц)
- Звук входа в трубу: нисходящий свуп 800→100 Гц
- Исправлен баг: при рестарте старая музыкальная дорожка больше не накладывается на новую

### Управление
- Поддержка геймпада: левый стик, D-pad и кнопка A (Xbox-раскладка)
- Клавиша N — переход на следующий уровень (дебаг)
- Клавиша P — пауза/снятие паузы; стрелки или пробел тоже снимают паузу; фоновая музыка останавливается на паузе
- Клавиша M — включение/выключение мьюта фоновой музыки (аналог кнопки в правом верхнем углу)
- Все клавиши переведены на `e.code` вместо `e.key` — управление работает при любой раскладке клавиатуры

### Интерфейс

- Экран Game Over с итоговым счётом и подсказкой по рестарту
